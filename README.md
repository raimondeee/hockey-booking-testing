# Hockey Booking — Homepage Preview

Static preview of the continuous-scroll homepage for Ben Stadey Hockey Training.

**Live preview (after enabling GitHub Pages):**  
https://raimondeee.github.io/hockey-booking-testing/

## Enable GitHub Pages

1. Open this repo on GitHub → **Settings** → **Pages**
2. Under **Build and deployment**, set **Source** to **Deploy from a branch**
3. Branch: **main** → folder **/ (root)** → **Save**
4. Wait 1–2 minutes, then open the URL above

No build step required — `index.html` is served as-is.

## Note

This is a **layout preview only**. Booking, calendar, and payments live in the main `hockey-booking` app on Render.
